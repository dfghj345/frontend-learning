### 0.css简介

css是层叠样式表的简称,用于美化html,布局网页

### 1.1 css语法规范

css规则由两个主要的部分组成:选择器以及一条或多条声明

例子:

```css
<head>
	<style>
		/*给谁改样式{改什么样式}*/
		p{color:red;font-size:12px;}
	</style>
</head>
<body>
	<p>有点意思</p>
</body>
```

#### 代码风格

**1.样式格式书写**

(1)紧凑格式

```
p {color: red;font-size: 12px;}
```



(2)展开式风格 :推荐

```
p {
	color: red;
	font-size: 12px;
}
```



**2.样式大小写**

(1)属性字母小写 :推荐

```
p {
	color: red;
}
```

(2)属性字母大写

```
P {
	COLOR: RED;
}
```

**3.空格规范**
(1) 属性值前面,冒号后面,保留一个空格

(2) 选择器与大括号中间保留空格



### 1.2 css选择器

选择器的作用就是根据不同需求选择不同标签



#### 选择器分类

选择器分为基础选择器和复合选择器两个大类

基础选择器是由单个选择器组成的

复合选择器包括:标签选择器,类选择器,id选择器和通配选择器



#### 标签选择器

标签选择器是指由html标签名称作为选择器,按标签名称分类,为页面中某一类标签指定统一的css样式

**语法**

```
标签名 {
	属性1: 属性值1;
	属性2: 属性值2;
}
```

#### 类选择器

单独选择一个或某几个标签,可以使用类选择器

**语法**

```
.类名{
	属性: 属性值1;
}
```

结构需要用class属性来调用class属性

例如

```css
<head>
	<style>
		.red {
    		color: red;
		}
	</style>
</head>
<body>
	<p class='red'> 这是一行文字 </p>
	<p> 这是一行文字 </p>
</body>
```

**多类名**

一个标签可以指定多个类名

```
<p class="red front20">亚瑟</p>
```

多个类名之间必须用空格隔开

(1) 可以把一些标签元素相同的样式共同的部分放到一个类里面

(2) 可以调用公共类,然后再调用自己独有的类



#### id选择器

id选择器可以为标有特定id的html元素指定特定的样式

html元素以id属性来设置id选择器,css中id选择器以#来定义

```
#id名 {
	属性1: 属性值1;
	...
}
```

例如,将id为nav元素中的内容设置为红色

```
#nav {
	color: red;
}
```

id属性只能在html文档中出现一次,一般搭配js使用

#### 通配符选择器

在css中,通配符选择器使用 "*" 定义,它表示选取页面的所有标签

```
* {
	属性1: 属性值;
}
```

通配符选择器不需要调用,自动给所有元素使用样式

```
* {
	marhin: 0;
	padding: 0;
}
```



### 1.3 css字体属性

css fonts(字体)属性用于定义字体系列,大小,粗细和文字样式



#### 字体系列

css使用 font-family属性定义文本的字体系列

```
p { font-family:"微软雅黑";}
p {font-family: Arial,"Microsoft yahei",微软雅黑;}
```

各种字体之间必须使用英语状态下的逗号隔开

一般情况下,如果有空格隔开的多个单词组成的字体,加引号

尽量使用系统默认字体 

```
body {font-family:'Microsoft YaHei',tahoma,arial,'Hiragino Sans GB'}
```

#### 字体大小

css使用front-size属性定义字体大小

```
p {
	font-size: 20px;
}
```

px(像素)大小是我们网页的最常见的单位

谷歌浏览器默认的文字大小16px

标题标签需要单独指定像素大小



#### 字体属性

css使用font-weight属性来定义字体属性

| 值                          | 描述                                                        |
| :-------------------------- | :---------------------------------------------------------- |
| normal                      | 默认值。定义标准的字符。                                    |
| bold                        | 定义粗体字符。                                              |
| bolder                      | 定义更粗的字符。                                            |
| lighter                     | 定义更细的字符。                                            |
| 100-900 | 定义由粗到细的字符。400 等同于 normal，而 700 等同于 bold。 |
| inherit                     | 规定应该从父元素继承字体的粗细。                            |



#### 文字样式

css使用font-style属性设置文本风格

```
p {
	font-style:normal|italic;
}
```

| 值      | 描述                                   |
| :------ | :------------------------------------- |
| normal  | 默认值。浏览器显示一个标准的字体样式。 |
| italic  | 浏览器会显示一个斜体的字体样式。       |
| oblique | 浏览器会显示一个倾斜的字体样式。       |
| inherit | 规定应该从父元素继承字体样式。         |



#### 复合属性

字体属性可以把文字样式综合来写

格式

```
body {
	font: font-style font-weight font-size/line-height font-fanily
}
```

注意,不能颠倒顺序,如` font: italic 400 16px 'Microsoft yahei';` 

必须保留font-size和font-family



### 1.4 文本属性

css test(文本)属性可定义文本的外观,比如文本的颜色,对其文本,装饰文本,文本缩进,行间距等



#### 文本颜色

color属性用于定义文本的颜色

```
p {
	color: red;
}
```

| 值           | 描述                                               |
| :----------- | :------------------------------------------------- |
| *color_name* | 规定颜色值为颜色名称的颜色（比如 red）。           |
| *hex_number* | 规定颜色值为十六进制值的颜色（比如 #ff0000）。     |
| *rgb_number* | 规定颜色值为 rgb 代码的颜色（比如 rgb(255,0,0)）。 |
| inherit      | 规定应该从父元素继承颜色。                         |

开发中常用16进制



#### 对齐文本

`text-align` 属性规定元素中的文本的水平对齐方式。

```
p {
	/* 本质是让盒子里的文字区中对齐*/
	text-align:center
}
```

| 值      | 描述                                       |
| :------ | :----------------------------------------- |
| left    | 把文本排列到左边。默认值：由浏览器决定。   |
| right   | 把文本排列到右边。                         |
| center  | 把文本排列到中间。                         |
| justify | 实现两端对齐文本效果。                     |
| inherit | 规定应该从父元素继承 text-align 属性的值。 |



#### 装饰文本

`text-decoration` 属性规定添加到文本的修饰。可以给文本添加下划线,删除线,上划线等

```
p {
	test-decoration: underline;
}
```

| 值           | 描述                     |
| :----------- | :----------------------- |
| none         | 默认                     |
| underline    | 下划线,链接 a 自带下划线 |
| overline     | 上划线                   |
| line-through | 删除线                   |

none可以取消 a 链接的下划线



#### 文本缩进

test-indent属性用来定义第一行的缩进

```
p {
	test-indent:20px;
	/*如果写的是2em,则缩进当前元素的两个字的距离*/
}
```

| 值       | 描述                               |
| :------- | :--------------------------------- |
| *length* | 定义固定的缩进。默认值：0。        |
| *%*      | 定义基于父元素宽度的百分比的缩进。 |



#### 行间距

line-height属性用于设置行间的距离(行高),可以控制文字与文字的距离

```
p {
	line-height:16px;
}
```

| 值       | 描述                                                 |
| :------- | :--------------------------------------------------- |
| normal   | 默认。设置合理的行间距。                             |
| *number* | 设置数字，此数字会与当前的字体尺寸相乘来设置行间距。 |
| *length* | 设置固定的行间距。                                   |
| *%*      | 基于当前字体尺寸的百分比行间距。                     |



### 1.5 样式表

#### 内部样式表

内部样式表（内嵌样式表）是写到html页面内部.是将所有的CSS代码抽取出来，单独放到一个`<style>`标签中。
```
<style>
	p {
		color: red;
		font-size: 12px;
	}
</style>
```

`<style>`标签理论上可以放在HTML文档的任何地方，但一般会放在文档的`<head>`标签中
通过此种方式，可以方便控制当前整个页面中的元素样式设置



#### 行内样式表

行内样式表（内联样式表）是在元素标签内部的style属性中设定CSS样式。适合于修改简单样式
```
<p style="color:red；font-size:12px;">青春不常在</p>
```
style其实就是标签的属性
在双引号中间，写法要符合CSS规范
可以控制当前的标签设置样式



#### 外部样式表

实际开发都是外部样式表.适合于样式比较多的情况.核心是：样式单独写到CSS文件中，之后把CSS文件引入到 HTML 页面中使用.

引入外部样式表分为两步：
1.新建一个后缀名为.css的样式文件，把所有CSS代码都放入此文件中。
2.在HTML页面中，使用`<link>`标签引I入这个文件。

```
<link rel="stylesheet" href="css文件路径">
```
